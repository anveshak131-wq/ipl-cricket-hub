Required CSV Format:

  • Columns: Name, Role, Age, Nationality, Batting Style, Bowling Style, Image
  • Role: Batsman, Bowler, All-rounder, Wicket-keeper
  • Age: Numeric value
  • Nationality: Country name (use "Indian" for Indian players)
  • Batting Style: Right-handed, Left-handed
  • Bowling Style: Right-arm fast, Left-arm spin, etc. (use "-" if not applicable)
  • Image: Image filename (e.g., virat_kohli.png)
  • Foreign players will be automatically marked with an indicator badge
